# Tutorial em PowerPoint — MinhaPrimeiraApi

## Como gerar o PowerPoint automaticamente

1. **Instale o Python** (se ainda não tiver): [python.org](https://www.python.org/downloads/) — marque a opção "Add Python to PATH".
2. **Instale a biblioteca python-pptx** no terminal:
   ```bash
   pip install python-pptx
   ```
3. **Execute o script** na pasta `apresentacao`:
   ```bash
   cd apresentacao
   python gerar_apresentacao.py
   ```
4. O arquivo **MinhaPrimeiraApi-Tutorial-Aula.pptx** será criado nesta mesma pasta. Abra no PowerPoint ou no Google Apresentações.

**Se você não tiver Python instalado:** use o arquivo **Conteudo-Slides-Tutorial.md** para copiar o texto de cada slide e montar o PowerPoint manualmente (ou em outro formato).

## Conteúdo da apresentação

- **Slide 1:** Capa (título e subtítulo)
- **Slides 2–3:** Objetivos e visão geral do projeto
- **Slides 4–5:** Docker (o que é, conceitos)
- **Slides 6–7:** MySQL e Entity Framework Core
- **Slide 8:** Fluxo da aplicação
- **Slides 9–11:** Passo a passo (pré-requisitos, executar com Docker, testar)
- **Slides 12–13:** Endpoints e estrutura do projeto
- **Slide 14:** Executar sem Docker (opcional)
- **Slide 15:** Referências e dúvidas

As **notas do orador** estão preenchidas em várias slides; use a visão “Notas” no PowerPoint para vê-las durante a apresentação.
